## 🎭 YouTube Clients

> Privacy-friendly and feature-enhanced alternatives to the official YouTube app.

- 🎭 [LibreTube](https://github.com/libre-tube/LibreTube) - Privacy-focused YouTube client using the Piped API; no Google account required.
- 🎭 [NewPipe](https://github.com/TeamNewPipe/NewPipe) - Lightweight client with background play, downloads, and no tracking.
- 🎭 [Morphe](https://github.com/MorpheApp) - Customizable YouTube client with a themeable interface.
- 🎭 [ReVanced](https://github.com/ReVanced) - YouTube mod featuring ad-blocking, SponsorBlock, and return of dislike counts.
- 🎭 [FreeTube](https://github.com/FreeTubeApp/FreeTube) - Desktop YouTube client that stores subscriptions locally without a Google account.
- 🎭 [SmartTube](https://github.com/yuliskov/SmartTube) - Ad-free YouTube client optimized for Android TV and set-top boxes.
- 🎭 [Materialious](https://github.com/Materialious/Materialious) - Material You-designed web YouTube client powered by the Piped API.
